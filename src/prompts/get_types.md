# Write an API client

## Task

* Output the types for requests and responses in this API.

## Response format

* Your response must contain only Rust code.
* If you encounter any problems, you must finish your reply with a block comment that starts with "Problems", for example:
  ```rust
  /*
  Problems:
  
  * Could not find a definition for enum Foo, which is mentioned in the struct Bar.
  
  */
  ```
